# Appointly API (minimal starter)

Quick starter for the API using Express + Mongoose.

Steps to run locally:

1. Install dependencies:

```bash
npm install
```

2. Start MongoDB locally (e.g., `mongod` or your MongoDB service).

3. Copy `.env` and update secrets if needed. Then run:

```bash
node app.js
```

Server will run on the `PORT` from `.env` (default 5000).
